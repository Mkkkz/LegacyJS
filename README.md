# LegacyJS Portfolio

Site de portfólio simples em Next.js para Vercel.

Discord configurado:
https://discord.gg/bpJWJQzT7p

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy na Vercel

Use:
- Framework: Next.js
- Build Command: npm run build
- Output Directory: Next.js default
- Install Command: npm install
